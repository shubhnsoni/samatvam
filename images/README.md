# Images & Videos Folder Structure

Place your media files here. They'll be deployed alongside the site.

```
images/
  founders/     → Deepti & Hritwik photos
  stories/      → Before/after transformation photos
  pillars/      → Restore, Nourish, Move section images
  branding/     → Logo, favicon, social share images

videos/
  courses/      → Course lesson videos (served by hosting provider)
  testimonials/ → Client testimonial videos
```

## Naming Convention
- `deepti-portrait.jpg`, `hritwik-portrait.jpg`
- `myraa-before.jpg`, `myraa-after.jpg`
- `restore-hero.jpg`, `nourish-hero.jpg`, `move-hero.jpg`
- `course-1-lesson-1.mp4`

## Notes
- Keep images under 500KB each (compress with TinyPNG)
- Videos: MP4 format, H.264 codec recommended
- Logo: SVG preferred, or PNG with transparent background
- Admin can also upload via admin panel → Supabase Storage (1GB free)
